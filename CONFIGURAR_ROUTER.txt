================================================================================
TROUBLESHOOTING
================================================================================

ERRO 1: "Nao consigo acessar 192.168.1.1"

  Solucao:
  1. Certifique-se que esta conectado ao Wi-Fi do router
  2. Tente: http://routerlogin.com (alguns routers usam isso)
  3. Reinicie o router (desligue por 30 segundos)
  4. Procure a etiqueta no fundo do router para o IP correto

ERRO 2: "Nao vejo "Port Forwarding" no router"

  Solucao:
  1. Procure em "Advanced Settings" ou "Configuracoes Avancadas"
  2. Procure "NAT" ou "Virtual Server"
  3. Seu router pode ter outro nome para isso
  4. Tente procurar no manual do router (marca+modelo ou logo)

ERRO 3: "Configurei mas ainda nao funciona"

  Solucao:
  1. Aguarde 5-10 minutos (demora para propagar)
  2. Limpe cache do navegador: Ctrl+Shift+Del
  3. Tente em outra rede (celular 4G) para testar
  4. Verifique se Flask/Nginx ainda estao rodando:
     No terminal: netstat -ano | Select-String ":5000|:8080"

ERRO 4: "Flask/Nginx pararam de rodar"

  Solucao:
  1. Reinicie ambos:
     cd C:\nginx && C:\nginx\nginx.exe -c conf/nginx.conf (em novo terminal)
     cd "C:\Users\Gabriela Resende\Documents\Plataforma ON" && python app.py
  2. Verifique os logs
